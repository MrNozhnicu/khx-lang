## 🚀 Главные нововведения

### 1. KHX CLI - Инструмент командной строки

Теперь KHX можно использовать прямо из терминала!

#### Установка
```bash
python khx_cli_installer.py
```

#### Основные команды
- `khx script.khx` - Запустить программу
- `khx --repl` - Интерактивный режим
- `khx --new project` - Создать проект
- `khx --check file` - Проверить синтаксис
- `khx --examples` - Показать примеры
- `khx --docs` - Открыть документацию
- `khx --version` - Показать версию
- `khx --help` - Справка

#### Интерактивный REPL
```bash
$ khx --repl
KHX REPL v5.2
khx> print("Hello!")
Hello!
khx> let x = 10
khx> print(x * 2)
20
```

#### Создание проектов
```bash
$ khx --new my_app
✅ Проект создан: my_app/
   - src/main.khx
   - README.md
```

## 🎯 Примеры использования

### Быстрый старт

```bash
# Установить CLI
python khx_cli_installer.py

# Попробовать REPL
khx --repl

# Создать проект
khx --new hello_app
cd hello_app
khx src/main.khx
```

### Разработка

```bash
# Создать проект
khx --new todo_app
cd todo_app

# Редактировать
nano src/main.khx

# Проверить
khx --check src/main.khx

# Запустить
khx src/main.khx
```

### Обучение

```bash
# Показать примеры
khx --examples

# Запустить пример
khx examples/calculator.khx

# Открыть документацию
khx --docs
```

## 🔧 Технические детали

### Поддерживаемые платформы

- ✅ Windows (7, 8, 10, 11)
- ✅ Linux (Ubuntu, Debian, Fedora, Arch и др.)
- ✅ macOS (10.14+)

### Требования

- Python 3.8+
- PyQt5 (для GUI)
- Опционально: requests (для Network модуля)

### Установка CLI

**Windows**:
```batch
python khx_cli_installer.py
```
Добавит `khx.bat` в PATH

**Linux/macOS**:
```bash
python khx_cli_installer.py
```
Добавит `khx` в `~/.local/bin` и обновит `.bashrc`/`.zshrc`
