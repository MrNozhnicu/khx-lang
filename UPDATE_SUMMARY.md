## 🎯 Примеры новых возможностей

### Data Science
```khx
// Сортировка и корреляция
let df = create_dataframe(data)
df = df.sort_by("age", true)
let corr = df.corr("height", "weight")
print("Корреляция:", corr)
```

### Mobile Development
```khx
// Хранилище и вибрация
let app = create_mobile_app("MyApp")
app.save_data("score", "1000")
app.vibrate(200)
let device = app.get_device_info()
```

### Game Engine
```khx
// Враги и очки
let game = create_game(800, 600, "Space Game")
game.spawn_enemy(400, 100, "boss")
game.add_score(100)
game.set_gravity(0.5)
```

### Machine Learning
```khx
// NLP анализ
let nlp = create_nlp()
let keywords = nlp.extract_keywords(text, 5)
let similarity = nlp.text_similarity(text1, text2)
```

### Audio
```khx
// Микшер и эффекты
let audio = load_audio("music.mp3")
audio.fade_in(2.0)
audio.apply_effect("reverb", 0.7)
audio.export("output", "wav")
```
